Welcome to Map Mainia V3.0

In this game you will be tasked with finding 10 locations by zooming into the map depending on hints given to you by the drop down menus.

Each location has two hints each in order to the respective location.

Example : hint #3 in Hint Set One and Hint # 3 in Hint Set Two both give information to find location 3.

When the location is found, the respective hins will disappear from the list

You will be awarded with 10pts for each location that you find for a grand total of 100pts.

good luck!!!
